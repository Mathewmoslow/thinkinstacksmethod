Thin In Stacks Method

Color-coded for quick navigation

1. Identify Question Type
Assess: gather Priority: first Do: action Eval: outcome Order: sequence
Stack of Four (Priority)
1. A-B-C-D
2. Safety (PHYSICAL only)
3. Maslow: nutrition, sleep, pain, elimination
4. Assess→Act
"Always Be Caring Daily"
Psychosocial = After Stack #4
By Question Format
Single: Stop at 1st Stack match
SATA: Check all → Pick all matches
Order: Pick all → Sort by Stack
Ordered Response Process:
1. ID all correct (like SATA)
2. Arrange: ABCD→Safety→Maslow
Quick Rules
Assess: Baseline > Causes
Implement: Least → Most invasive
Delegate: Stable=Yes, Unstable=No
Evaluate: Own words = Got it
Maslow Q's: nutrition, sleep, pain, bathroom = Stack #3
*Urine retention/impaction → Stack #2
Final Filters
• Always/Never = Red flag
• Two similar = Both out
• Covers all = Winner
• Changed answer? Go back!
• First instinct = Best
TIE-BREAKERS
Same Stack Level?
1. Assessment about tied items → WINS
Ex: "Assess patterns" beats "Give food"
2. No assessment? → Use urgency:
Hours: Glucose, Urine
Days: Pain, Fluids
Weeks: Food, Sleep, Activity
